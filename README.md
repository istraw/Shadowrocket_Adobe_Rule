# Shadowrocket屏蔽Adobe的网络验证
### ⚠️ 开启VPN后，全部流量会走VPN，原防火墙和hosts设置的屏蔽会失效！需在VPN设置里加上Adobe的屏蔽名单。
最后更新时间：2026-03-11

个人使用，遇到屏蔽失效时会更新。

如有建议，可以在Issues中提交。

## 使用方法
1. 复制以下规则链接，在 小火箭 -> 配置 -> 远程文件 中添加
2. 小火箭 -> 配置 -> 模块，右上角，添加规则（模块化规则链接）
3. 完成
