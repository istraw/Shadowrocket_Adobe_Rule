# Shadowrocket屏蔽Adobe的网络验证
### ⚠️ 温馨提示： 开启VPN后防火墙和hosts屏蔽会失效！请在你的VPN路由规则里手动加上Adobe的屏蔽名单，否则软件可能会提示由于网络环境导致的激活失败。
最后更新时间：2026-03-11

个人使用，遇到屏蔽失效时会更新

如有建议，可以在Issues中提交

## 使用方法
1. 复制以下规则链接，在 小火箭 -> 配置 -> 远程文件 中添加
2. 小火箭 -> 配置 -> 模块，右上角，添加规则（模块化规则链接）
3. 完成
